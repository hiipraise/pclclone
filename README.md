# PCL Clone

This project is a frontend clone of the PCL application built with React.

## Features

- Responsive UI
- Modern React architecture
- Component-based structure

## Getting Started

1. **Install dependencies:**
    ```bash
    npm install
    ```
2. **Start the development server:**
    ```bash
    npm start
    ```
